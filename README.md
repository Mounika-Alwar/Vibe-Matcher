
# Vibe Matcher Notebook: Prototype AI-Powered Fashion Recommendation

## Overview
This notebook demonstrates the prototype of a Vibe Matcher system, which converts product descriptions and user queries into vector embeddings and computes cosine similarity for recommendation.

## Contents
- Creation of a small mock dataset with products, descriptions, vibes, and categories
- Generating embeddings using SentenceTransformer
- Building a FAISS index for similarity search
- Querying top-k product matches with similarity scoring
- Handling fallback cases
- Evaluating queries and latency

## Features
- Embeddings-based similarity matching
- Top-k ranking
- Fallback suggestions for weak matches
- Evaluation metrics and latency measurement
